Testing jquery, learning by doing! A wonderful aspect of jQuery is the fact that most of the methods return an object that you can then use to call another method. This allows you to do command chaining where you can perform multiple methods on the same set of elements. This can be very useful and can save a lot of time.

Testing jquery, learning by doing! Chaining gives us the ability to run multiple jQuery methods on a single element while using only a single statement. $('p').removeClass("make-border").addClass("make-red"); This can also be written like shown below and is still considered valid format, but for testing stick to using the single line version. $('p').removeClass("make-border") .addClass("make-red");
